# Aerobraking
Team Tumbleweed bus aerobraking calculations

## Prerequisites
`python -m pip install --upgrade pip`

`pip install tqdm`

`pip install numpy`

`pip install matplotlib`

For creating matplotlib animations you need [ffmpeg](https://www.wikihow.com/Install-FFmpeg-on-Windows)

### Extra

`pip install pandas`
